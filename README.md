# 🎮 Tic-Tac-Toe Game

A simple and interactive web-based **Tic-Tac-Toe** game built using **HTML**, **CSS**, and **JavaScript**. Two players can take turns marking "O" and "X" on a 3×3 grid, with automatic win or draw detection.

---

## 📌 Features

- ✅ Two-player mode
- 🧠 Win and draw detection
- 🔄 Reset & New Game button
- 🎨 Responsive and clean UI
- 🧹 Beginner-friendly, modular code

---

## 🛠️ Technologies Used

| Frontend     | Purpose                          |
|--------------|----------------------------------|
| HTML         | Markup and structure             |
| CSS          | Styling and layout               |
| JavaScript   | Game logic and user interaction  |

---

## 🚀 How to Run

### ✅ Option 1: Using Live Server (Recommended)
1. Install **VS Code** and the **Live Server** extension.
2. Open the project folder in VS Code.
3. Right-click on `index.html` → **Open with Live Server**.
4. The game will open in your default browser.

### 🔍 Option 2: Without Live Server
1. Open the project folder.
2. Double-click on `index.html`.
3. The game will open in your browser.
